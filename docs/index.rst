psico: PyMOL Scripts Collection
================================

psico is a Python package that extends `PyMOL <https://pymol.org>`_ with a large
set of useful commands and helper functions.

The upstream wiki page is available at `pymolwiki: psico <https://pymolwiki.org/index.php/psico>`_.
This site provides installation instructions, quick usage hints, and a
reference API generated from the source code.
